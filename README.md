# Render-Distance-Keybinds
Adds the render distance shortcut back on Minecraft Version 1.19+ and sets it to a single keybind [ to decrease and ] to increase

Main version is for Minecraft Java 26.2, Other branches are for other versions,

1.19 works with version 1.19 all the way to 1.21.8

1.21.9 works for 1.21.9 to 1.21.11

&

26.1 works for 26.1, 26.1.1, & 26.1.2

26.2 (Main Branch) Works only for 26.2
